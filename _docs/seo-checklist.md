# Checklist SEO per Autori - Rosso di Sera

Prima di pubblicare, verifica questi punti:

## Titolo e Description
- [ ] Il **Titolo SEO** (se diverso dal titolo) contiene la parola chiave principale?
- [ ] Il Titolo SEO è max 60 caratteri?
- [ ] La **Meta Description** è tra 120-155 caratteri?
- [ ] La Description contiene la parola chiave?
- [ ] La Description è una CTA che invoglia al click, non un riassunto noioso?

## Contenuto
- [ ] La parola chiave appare nel **primo paragrafo** (prime 100 parole)?
- [ ] Almeno un **H2** contiene la parola chiave (o variante)?
- [ ] L'articolo ha almeno **300 parole**? (ideale: 800+)
- [ ] Ci sono **link interni** ad altri articoli del blog?
- [ ] Le frasi sono brevi e i paragrafi ariosi?

## Immagini
- [ ] L'immagine principale è almeno **1200x630px**?
- [ ] L'**Alt Text** descrive cosa si vede nell'immagine?
- [ ] L'Alt Text NON è uguale al titolo dell'articolo?

## URL
- [ ] Il permalink (slug) è breve e contiene la keyword?
- [ ] Niente date, numeri o caratteri speciali nello slug?

---

## Esempi

**Parola chiave:** "crisi climatica Italia"

❌ Titolo SEO sbagliato: "Riflessioni sul nostro pianeta"
✅ Titolo SEO corretto: "Crisi climatica in Italia: cosa sta succedendo"

❌ Description sbagliata: "In questo articolo parliamo di clima."
✅ Description corretta: "La crisi climatica in Italia accelera: ecco i dati 2024 e cosa possiamo fare per invertire la rotta."

❌ Alt text sbagliato: "Immagine articolo crisi climatica"
✅ Alt text corretto: "Ghiacciaio alpino in ritiro con rocce esposte e lago glaciale"
